# Put your procedures (SOPs) here

An SOP is your organization's official rule document for how work is done. Drop your
SOP files into this folder — PDF or Word both work — and the workbench will cite the
exact document and section when it writes or checks your deliverables.

## How to add an SOP
1. Copy your SOP files into this folder.
2. That's it. The assistant reads them automatically when it needs a rule.

## If this folder is empty
The workbench still works. When a rule would normally come from an SOP, the assistant
uses general good-practice knowledge and tells you which SOP to add here so that future
answers can cite a specific document and section.

## Good to know
- Files here are never modified. The workbench only reads them.
- The assistant never invents an SOP name or section number. It cites only what it can
  actually find in a file in this folder.
- Add the current, approved version of each SOP. Replace old versions when they change.
